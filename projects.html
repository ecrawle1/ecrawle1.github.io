Automated Batch Posting, AppXtender Integration & GLRSLCT Selections

Built Oracle SQL–driven automations for nightly batch posting and downstream processes, integrated with AppXtender, and used GLRSLCT population selections to power Cognos letters and targeted communications.

Stack
Oracle SQL, Ellucian Banner 9, AppXtender, IBM Cognos (GLRSLCT)
Role
Design & implementation of batch jobs, data flows, and letter generation; SOPs & handoffs
Outcome
Faster postings, fewer manual steps, traceable document indexing, and consistent letter output.
  • Designed and tuned SQL used by scheduled jobs for **automated batch posting** and follow-on processes.
  • Integrated with **AppXtender** for document indexing/attachments tied to student and transaction records.
  • Leveraged **GLRSLCT** population selections to feed **Cognos** for letter creation and targeted outreach.
  • Added exception handling, run logs, and SOPs to improve auditability and reduce support time.
